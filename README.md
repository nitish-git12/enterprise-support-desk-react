# Enterprise Support Desk (React)

This project is the foundation for an enterprise-style support desk application built using modern React tooling.

## Tech Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS

## Current Status
- Project bootstrapped with Vite
- Tailwind CSS configured
- Clean folder structure initialized

This project will be extended step-by-step to include authentication, routing, state management, and REST API integrations.
